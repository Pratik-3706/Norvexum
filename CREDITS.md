## Core Libraries Used
This project is built using the power of the Rust ecosystem and the following core crates:
*   [ratatui](https://github.com/ratatui/ratatui) - Terminal User Interface builder
*   [crossterm](https://github.com/crossterm-rs/crossterm) - Cross-platform terminal raw mode and event handler
*   [tokio](https://github.com/tokio-rs/tokio) - Asynchronous waker and runtime
*   [reqwest](https://github.com/seanmonstar/reqwest) - HTTP client with TLS support
*   [serde](https://github.com/serde-rs/serde) - Serialization and deserialization framework
*   [scraper](https://github.com/causal-agent/scraper) - HTML scraping and parsing engine
*   [html2text](https://github.com/zg/html2text) - HTML-to-text formatter for TUI reading
*   [globset](https://github.com/BurntSushi/ripgrep/tree/master/crates/globset) - Glob matching logic for file lookups
*   [walkdir](https://github.com/BurntSushi/walkdir) - Recursive directory walker
*   [similar](https://github.com/mitsuhiko/similar) - Diffing library for code edits
*   [arboard](https://github.com/1aim/arboard) - System clipboard integration
